# Machine Learning Project: Predictive Modeling & OCR

This repository contains a Google Colab notebook that demonstrates multiple data science workflows, including regression, classification, and optical character recognition (OCR).  
The project showcases end-to-end steps: data preprocessing, model training, evaluation, and visualization.

## 🚀 Features
- Data preprocessing (cleaning, normalization, feature engineering)
- Regression models (Linear Regression, Random Forest Regressor)
- Classification models (Logistic Regression, Decision Trees, SVM)
- Optical Character Recognition (OCR) using OpenCV and Tesseract
- Model evaluation with R² Score, Accuracy, Precision, Recall, F1-score
- Visualizations with Matplotlib and Seaborn

## 📂 Project Structure
- `notebook.ipynb` → Main Colab notebook
- `requirements.txt` → Dependencies
- `data/` → Sample datasets (optional)
